### CONTROLES

JUGADOR IZQUIERDA : 
	Tecla W (movimiento hacia arriba)
	Tecla S (movimiento hacia abajo)

JUGADOR DERECHA:
	Tecla Flechita ARRIBA(movimiento hacia arriba)
	Tecla Flechita ABAJO (movimiento hacia abajo)

### OBJETIVO
	Sumar puntos al evitar que la bola toque el borde.
